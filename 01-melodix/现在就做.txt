2. 找到你的应用
3. 点击右边的菜单图标 ⋮
4. 选择 "Settings"

确认以下配置：

    ┌─────────────────────────────────────────┐
    │ Repository: [你的 GitHub 仓库]            │
    │ Branch: main                            │
    │ Main file path: streamlit_app.py        │
    └─────────────────────────────────────────┘

5. 展开 "Advanced settings"：
    
    ┌─────────────────────────────────────────┐
    │ Python version: 3.11                    │
    │ Requirements file: (留空自动检测)         │
    └─────────────────────────────────────────┘

6. 点击 "Save"
